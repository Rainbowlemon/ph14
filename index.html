Why would I want this framework?

If you're working on a project that's expecting a designer later down the line, sometimes you don't want it to look absolutely terrible until then. Adding a framework like Bootstrap means having to add classes and <div>s to your code, which may all get chucked out if the designer/front-end developer prefers a different front-end framwork.

By using PH14, you can add some basic styles to your semantic HTML which can be cleaned up later down the line.
